GetCTStream
===========

Python script pro parasovani streamu živého vysíláni programu ČT24.

Instalace
---------

1. Samostatný skript

    python2 get_ct_stream.py

2. Kodi plugin

    See [Kodi/KODI_README.md](https://github.com/kunesj/kodi-plugin.video.streamct/blob/master/Kodi/KODI_README.md)
